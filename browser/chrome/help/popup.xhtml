Controlando ventanas emergentes

Este documento explica todas las &pref.plural; disponibles en &brandFullName; para controlar las ventanas emergentes.

En esta sección:
  • ¿Qué son las ventanas emergentes?
  • &pref.pluralCaps; del bloqueador de ventanas

¿Qué son las ventanas emergentes?

Las ventanas emergentes, también conocidas por su nombre en inglés popups, son ventanas que aparecen automáticamente sin permiso. Estas ventanas pueden tener un tamaño variable, aunque usualmente no cubren toda la pantalla. Algunas ventanas emergentes se abren en la parte superior de la ventana de &brandShortName;, mientras que otras aparecen debajo de &brandShortName; (popunders).

&brandShortName; permite controlar las ventanas emergentes y popunders con el Panel de características web en &pref.pluralCaps;. El bloqueo de las ventanas emergentes está activado por defecto, en consecuencia el usuario no debe procuparse por poner en funcionamiento el bloqueo de ventamas emergentes en &brandShortName;.

Cuando una ventana emergente ha sido bloqueda, en &brandShortName; aparece una barra de información, como también un ícono en la barra de estado. Cuando se hace clic o en el el botón Opciones en la barra de información o en el ícono de la barra de estado, apacere un menú con las siguientes opciones:

  • Permitir/Bloquear ventanas para este sitio
  • Editar opciones del bloqueo de ventanas emergentes
  • No mostrar este mensaje (mensaje informativo) cuando se bloqueen ventanas emergentes
  • Mostrar

Bloqueando ventanas emergentes puede interferir con algunos sitios web: Algunos sitios web, incluyendo algunos sitios de Bancos, usan ventanas emergentes para importantes características. Bloqueando todas las ventanas emergentes inhabilita dichas características. Para permitir que sitios web específicos utilicen las ventanas emergentes, mientras que bloquea todos los otros, puede agregar sitios web específicos a la lista de los sitios permitidos.

El bloqueo de ventanas emergentes no siempre funciona: Aunque &brandShortName; bloquea la mayoría de las ventanas emergentes algunos sitios web las pueden mostrar usando métodos desconocidos inclusive cuando hayan sido bloquedas.

&pref.pluralCaps; del bloqueador de ventanas emergentes

Esta sección describe como usar el bloqueador de ventanas emergentes de &brandShortName;. Si aún no lo visualiza abra el panel de características web a través de los siguientes pasos:

  1. Seleccione &pref.menuPath;.
  2. Haga click en el panel Características web.

Desde este panel Ud. puede elegir las siguientes opciones:

  • Bloquear ventanas emergentes
  • Sitios permitidos: Ésta es una lista de los sitios web que pueden abrir ventanas emergentes. El diálogo tiene las siguientes opciones:
    • Permitir: Haga clic para agregar un sitio web a la lista de excepciones.
    • Remover Sitio: Haga clic para remover un sitio web de la lista de excepciones.
    • Remover todos los sitios: Haga clic para remover todos los sitios web de la lista de excepciones.

Nota: El bloqueo de sitios web no siempre funciona y puede interferir con algunos sitios web. Para mayor información acerca de el bloqueo de ventanas emergentes ver Qué son las ventanas emergentes.
