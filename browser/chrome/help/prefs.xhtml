Tipos de letra y colores

Haga clik en el botón Tipos de letra y colores... para cambiar la tipografía y colores predeterminados usados por &brandShortName;.

Tipos de letra para:
Normalmente, las páginas web son mostradas con el conjunto de tipos de letra predeterminados de su explorador o con una tipografía elegida por el autor de la página web.

Para cambiar la tipografía predeterminada:

  1. De la lista desplegable de Tipos de letra para, elija un conjunto de caracteres. Por caso, para elegir la tipografía predeterminada para el conjunto de caracteres Occidental (Romano), elija Occidental.
  2. Elija si el texto proporcional será serif (como "Times New Roman") o sans-serif (como "Arial"). Entonces especifique el tamaño de letra que desea para el texto proporcional.
  3. Especifique la tipografía para usar para las letras Serif, Sans-serif y Tamaño fijo. Puede además cambiar el tamaño para letras de tamaño fijo.

Puede además especificar la resolución de muestra, la cual es normalmente de 96 dpi.

Finalmente, puede elegir el tamaño mínimo de letra para ser usado en la pantalla. Ésto puede ser útil en algunos sitios que usan letras muy pequeñas que son apenas legibles

Preferencias de Conexión

Su organización o su proveedor de servicios de internet puede ofrecerle o requerirle que use un proxy. Un proxy actúa como un intermediario entre su computadora e Internet. Intercepta todos los pedidos hacia Internet para ver si puede satisfacer el pedico usando su caché. Los proxys son usados para mejorar la performance, filtrar pedidos y ocultar su computadora de Internet para mejorar la seguridad. Los proxys son a menudo usados como parte de un firewall corporativo.

Configurar Proxys para acceder a Internet
Conexión directa a Internet
Esta es la &pref.singular; predeterminada. Elija esta &pref.singular; si no desea usar un proxy.
Configuración manual de proxy
Elija esta &pref.singular; si no tiene una dirección de proxy (URL). Pregunte a su administrador de sistema por el nombre y número de puerto del servidor que corre el software de proxy para cada servicio de red e ingrese la información en los campos apropiados.
URL de configuración automática del proxy
Si su lugar de tranajo tiene un archivo de configuración de proxy, pregúnte al administrador del sistema por esa URL e introdúzcala aquí. Haga clic en Recargar para cargar la configuración.

&pref.pluralCaps; Avanzadas

El panel avanzado contiene muchas &pref.plural; que son poco probables que sean usadas por la mayoría de las personas pero son aún útiles y algunas veces son &pref.plural; críticas para algunas personas.

Accesibilidad

Mover el cursor del sistema con el foco o cambios de selección
Tambien conocido como Navegación con Cursor, esto especifica que si se muestra un cursor movible, permitiéndole seleccionar texto con el teclado. Algunas ayudas de accesibilida, tal como lectores de pantalla o ampliadores de pantalla, usan el cursor del sistema para determinar que area de la pantalla leer o amplifar. Puede setear este modo al presionar F7.

Iniciar busqueda al comenzar a tipear
Cuando esta &pref.singular; esta habilitada, &brandShortName; encontrarán la página web actual lo que tipee tal como lo tipee. Mientras este buscando texto en la página, la Barra de Búsqueda será automáticamente mostrada en la parte inferior de la ventana para mostrar información acerca de lo que haya encontrado.

Browsing

Redimencionar imágenes grandes para caber en la ventana de &brandShortName;
Esta característica es semejante a la vista en Internet Explorer. Hace que imagenes que son más grandes que la ventana, sean ajustadas a la misma así puede ver la imágen completa. Cliquee en la imágen para verla a tamaño completo.

Usar autodeslizamiento
Autodeslizamiento es una característica de utilidad que permite desplazar la página con solo presionar el botón medio del mouse (usualmente la rueda de desplazamiento) y mover el mouse arriba o abajo. Algunas personas encuentras esto molesto, por tanto el autodesplazamiento puede ser deshabilitao con esta &pref.singular;.

Usar desplazamiento suave
Desplazamiento suave es aún algo experimental en &brandShortName;, pero puede ser muy útil si lee muchas páginas largas. Normalmente, cuando presiona la tecla Avance de Página (Page Down), la vista salta directamente una página abajo. Con desplazamiento suave, se desliza hacia abajo mas suavemente, así esta efectivamente habilitado a ver cuánto se desplaza. ESto hace mas sencillo continuar la lectura desde donde estaba antes.

Navegación Tabulada ( Tabbed Browsing )

Abir enlace desde otras aplicaciones en:
Cuando otras aplicaciones en su computadora muestran una página web, &brandShortName; abre la página en la solapa o ventana más recientemente mostrada. Puede hacer que &brandShortName; abra la página en otro lugar cambiando esta &pref.singular;.

Forzar enlaces que abran nuevas ventanas abrir en:
&brandShortName; abre nuevas ventanas desde enlaces cuando los sitios web lo solicitan. Si prefiere usar solapas a medida que navegue, este comportamiento puede ser contraproductivo. Puede forzar &brandShortName; a abrir enlaces que serían normalmente abiertos en ventanas nuevas ser abiertos en otro lugar. Para ello, seleccione la tilde para esta &pref.singular; y elija el comportamiento que desee.

Ocultar la barra de solapas cuando solo un sitio web esté abierto
Si esta viendo solo una página web en una ventana de &brandShortName;, la barra de solapas no será mostrada. Destilde esta &pref.singular; para siempre mostrar la barra de solapas.

Seguridad

Usar SSL 2.0
Especifica si desea enviar y recivir información segura a travéz de SSL2 (Secure Sockets Layer Level 2), el protocolo estandar para transmisiones seguras. Todos los sitios web seguros soportan este protocolo.

Usar SSL 3.0
Especifica si desea enviar y recivir información segura a travéz de SL3 (Secure Sockets Layer Level 3), un protocolo que se estima es mas seguro que SSL2. Note que algunos sitios web pueden no soportar este protocolo.

Usar TLS 1.0
Especifica si desea enviar y recivir información segura a travéz de TLS (Transport Layer Security), un estandar de seguridad abierto semejante a SSL3 (Secure Sockets Layer). Note that some web sites may not support this protocol.

Certificados

Certificados ayudan a realizar encriptaciones y decriptaciones de conecciones a sitios seguros.

Seleccion del certificado del cliente
Cuando un sitio web solicita una conección segura, &brandShortName; predeterminad y automáticaamente usará un certificado apropiado. Si desea manualmente elegir un certificado (por ejemplo, si desea usar un cierto ripo de encriptación en lugar de la que es elegida automáticamente), elija la &pref.singular; "Preguntar Cada Vez" y tendrá el control completo de que sertificados usará mientras navega.

Validación

Validación se asegura que los certificados usados por &brandShortName; no son obsoletos.

CRL
&brandShortName; puede usar CRLs (tambien conocidos como Certificate Revocation Lists - Lista de Revocación de certificados) para asegurarse de qeu sus certificados no son inválidos. Si necesita añadir un CRL a &brandShortName; o desea ver información sobre los CRLs que tiene instalados, cliquee el botón Manejar CRLs....

OCSP
OCSP (Online Certificate Status Protocol - Protocolo en Linea de Estado de Certificados ) es una manera de validar sus certificados cada vez que sean vistos o usados. &brandShortName; no usa OCSP predeterminadamente, pero si desea puede habilitarlo, puede hacer ello aquí. Usted necesitará muy probablemente solamente cambiar eso si su entorno de Internet lo requiere.
