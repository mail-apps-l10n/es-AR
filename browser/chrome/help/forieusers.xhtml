Acerca de sus favoritos y opciones de IE

La primera vez que instale &brandShortName;, se importarán sus opciones existentes de Internet Explorer, incluyendo sus favoritos, cookies, contraseñas guardadas, y una variedad de datos. Ésto le ahorrará tiempo al personalizar &brandShortName; para que cubra sus necesidades.

Características de &brandShortName;

  • Navegación con pestañas: En lugar de abrir una ventana separada para cada sitio que visite, puede abrir múltiples sitios dentro de la misma ventana en diferentes pestañas. También se puede poner un grupo de pestañas como su página de inicio. Para mayor información vea Navegación con pestañas.
  • Controles para ventanas emergentes: Puede permitir o suprimir ventanas emergentes.
  • Administrador de cookies: Permite cambiar la forma en que &brandShortName; reaccionará al aceptar cookies.
  • Administrador de descargas: Organiza sus descargas guardándolas en una sola ventana para minimizar el desorden. Tiene acceso total a las estadísticas de descarga sin ventanas innecesarias para cada archivo.
  • Administrador de imágenes: Permite deshabilitar imágenes de ciertos sitios web o deshabilitarlas por completo. Ésto es útil si desea disminuir el tiempo que toma cargar un sitio web.

Atajos de teclado

Notará que Microsoft Internet Explorer y &brandShortName; comparten muchos atajos de teclado. Para una lista completa de atajos de teclado, vea la lista de atajos de teclado de &brandShortName;.
