Navegación con pestañas

La navegación con pestañas permite navegar la web de una forma más conveniente. Esta sección describe como usar las pestañas y como personalizarlas para hacer que la navegación de la web sea aún más rápida.

En esta sección:
  • Usando la navegación con pestañas
  • Creando y cargando pestañas
  • Personalizar la navegación con pestañas

Usando la navegación con pestañas

La navegación con pestañas le permite abrir pestañas, cada una mostrando una página web, en una sola ventana de &brandShortName;. Ésto libera espacio en su escritorio.

You can also "browse ahead" with tabbed browsing. If you are on a long web page, you can load links in new tabs. The linked web pages will load in tabs while you read the current page.

Creando y cargando pestañas

Para abrir una nueva pestaña, pulse &accelKey;+T, seleccione Archivo > Nueva pestaña, o haga doble-clic en un espacio vacío en la barra de pestañas.

Para cargar un enlace en una pestaña, puede hacer lo siguiente:

  • Haga clic con el botón central en el enlace. (Si tiene un ratón con rueda, hacer clic en la rueda es equivalente al clic central.)
  • Arrastre el enlace y suéltelo en un espacio vacío de la barra de pestañas. (Si hay una sola página web abierta, la barra de pestañas puede estar oculta. Ver Personalizar la navegación con pestañas para saber como cambiar ésto.)
  • Arrastre y suelte el enlace a una pestaña para abrir el enlace en esa pestaña.
  • Haga clic con el botón derecho enHaga &ctrlKey;, clic en el enlace y seleccione Abrir enlace en nueva pestaña desde el meú contextual.

Para abrir una URL de la barra de direcciones en una nueva pestaña, ingrese la URL y pulse &altKey;+&enterKey;.

Cerrando pestañas

Para cerrar la pestaña actual, pulse &accelKey;+W, haga clic en el botón Cerrar pestaña, haga clic con el botón central en la pestaña, o seleccione Archivo > Cerrar pestaña.

Para cerrar todas las pestañas menos la actual, haga clic con el botón derechohaga &ctrlKey;, clic en la pestaña y seleccione Cerrar las otras pestañas.

Atajos de teclado

Aunque es muy fácil seleccionar pestañas con el ratón, se pueden recorrer usando el teclado: use &ctrlKey;+Page Up (o &ctrlKey;+Tab) y &ctrlKey;+Page Down (o &ctrlKey;+&shiftKey;+Tab). También se puede seleccionar la primera pestaña presionando &accelKey;&altKey;+1, y así hasta la 9na pestaña con &accelKey;&altKey;+9.

Marcadores y la navegación con pestañas

Para marcar un grupo de pestañas:

  1. Pulse &accelKey;+D o seleccione Marcadores > Marcar esta página.
  2. Tilde el cuadrado Marcar todas las pestañas en una carpeta para guardar todas las pestañas de la ventana actual en una carpeta de marcadores.
  3. Para abrir marcadores en pestañas, abra una carpeta del menú de marcadores y seleccione Abrir en pestañas. También puede hacer clic con el botón central en la carpeta para abrir su contenido en pestañas. Los marcadores se abrirán en pestañas existentes, no en nuevas.

Página de inicio como pestañas

En lugar de usar solamente una página web como página de inicio, puede hacer que su página de inicio sea un grupo de pestañas. Seleccione sus sitios web favoritos y ábralos en pestañas en una ventana. Seleccione &pref.menuPath; y abra el panel General. En Página de inicio, pulse el botón Usar páginas actuales. Ahora cuando presione el botón Inicio, sus sitios web favoritos se cargarán con un solo clic.

Personalizando la navegación con pestañas

Para personalizar la navegación con pestañas, seleccione &pref.menuPath; y abra el panel Tabs. &pref.pluralCaps; para la navegación con pestañas pueden encontrarse bajo Navegación con pestañas.

Consejos útiles

  • Puede agregar un botón Nueva pestaña a la barra de herramientas para abrir fácilmente nuevas pestañas personalizando su barra de herramientas.
  • Aunque &brandShortName; ya provee las &pref.plural; de navegación con pestañas más ampliamente usadas, usted puede desear personalizar la navegacióon con pestañas aún más usando extensiones.
