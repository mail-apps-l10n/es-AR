Referencia de menú de &brandFullName;

&brandShortName;

Acerca de &brandFullName;

Muestra información sobre &brandShortName;, incluyendo la versión actual y una breve lista de créditos.

Preferencias...

Displays the &pref.pluralCaps; window, where you can change many &pref.plural; in &brandShortName;.

Salir de &brandShortName;

Cierra todas las ventanas abiertas de &brandShortName;, incluyendo cualquier descarga activa, y sale por completo. Si selecciona este ítem de menú con pestañas abiertas, ventanas o descargas, &brandShortName; pedirá confirmación para evitar salidas accidentales.

Archivo

Nueva ventana

Abre una ventana nueva de &brandShortName;.

Nueva pestaña

Abre una nueve pestaña del navegador en la ventana actual de &brandShortName;.

Abrir dirección...

Dirige el cursor a la barra de direcciones de &brandShortName; para poder escribir una nueva ubicación a visitar.

Abrir archivo...

Muestra un diálogo para abrir donde se puede seleccionar un archivo de sus discos locales o la red. Se pueden ver varios tipos de archivo en &brandShortName;, incluyendo archivos HTML/XML, imágenes, texto y más.

Cerrar (Ventana)

Cierra la ventana actual.

Cerrar pestaña

Cierra la pestaña actual y selecciona la que se encuentre más a la derecha. Este ítem de menú es visible solamente si hay abierta más de una pestaña.

Guardar página como...

Guarda la página que se está navegando actualmente. Se puede seleccionar la página completa (incluyendo los gráficos), solamente el código HTML, o el texto de la página solamente.

Enviar página...

Abre una ventana de redactar correo electrónico usando su programa predeterminado para poder enviar un mensaje con un enlace a la página actual. Esta característica está disponible solamente en Windows, Mac OS X y OS/2 por el momento.

Configuración de página...

Muestra un diálogo de configuración de página donde se pueden especificar las opciones de impresión como márgenes, encabezados y pies de página, y la orientación de la página.

Vista previa (Windows y Linux solamente)

Muestra una vista previa de como aparecerá impreso el documento actual. Refreja las opciones del diálogo de configuración de página y las intenciones del autor de la página.

Imprimir...

Muestra el diálogo de impresión, donde se puede especificar el número de copias impresas, etc. Haga clic en Aceptar para imprimir la página.

Importar...

Abre el asistente de importación, que permite importar &pref.plural;, marcadores, historial, contraseñas y otros datos de navegadores como Microsoft Internet Explorer, Netscape, Mozilla u Opera.

Trabajar sin conexión

Cambia entre los modos con conexión y sin conexión. El modo sin conexión permite ver páginas web que haya visitado previamente sin necesidad de conectarse a Internet.

Salir Salir

Cierra todas las ventanas abiertas en &brandShortName;, incluyendo las descargas activas, y sale de &brandShortName; por completo. Si selecciona este menú con pestañas, ventanas o descargas abiertas, &brandShortName; pedirá confirmación para evitar salidas accidentales.

Editar

Deshacer

Deshace la última acción de un campo de texto. Si decide posteriormente que no desea deshacer una acción, use el comando Rehacer.

Rehacer

Rehace el último comando Deshacer.

Cortar

Corta el texto seleccionado de un campo de texto y lo guarda en el portapapeles. Pegue el texto en cualquier lugar usando el comando Pegar.

Copiar

Copia el texto seleccionado al portapapeles. Pegue el texto en un campo de texto usando el comando Pegar.

Pegar

Pega el texto guardado en el portapapeles en un campo de texto.

Borrar

Borra el texto seleccionado de un campo de texto.

Seleccionar todo

Selecciona todo el texto y otros ítems en la página.

Buscar en esta página...

Muestra la barra de búsqueda en la parte inferior de la ventana del navegador. Ingrese el texto que desea buscar en la barra, y mientras tipea &brandShortName; irá buscando automáticamente y selecciona la primera ocurrencia del texto ingresado en la página. Haga clic en Buscar siguiente o Buscar anterior para busar el texto en otras ubicaciones en la página, y haga clic en Resaltar para resaltar todas las ocurrencias del texto en la página.

Buscar de nuevo

Busca la próxima ocurrencia del texto especificado al usar comando Buscar en esta página....

&pref.pluralCaps;

Muestra la ventana de &pref.pluralCaps;, donde se pueden cambiar las &pref.plural; de &brandShortName;.

Ver

Barras de herramientas

Barra de navegación

Cambia la visibilidad de la barra de navegación, que usualmente contiene botones estándar de navegación (Atrás, Adelante, Detener, etc.).

Barra de marcadores

Cambia la visibilidad de la barra de marcadores, que contiene marcadores a distintos sitios.

Personalizar...

Muestra el diálogo de personalización de barra de herramientas. Se pueden arrastrar y soltar botones a o desde las barras de herramientas aquí. También se puede hacer clic derechopulsar &ctrlKey;, clic en la barra de navegación y seleccionar Personalizar... para mostrar este diálogo.

Barra de estado

Cambia la visibilidad de la barra de estado, ubicada en la parte inferior de la ventana. La barra de estado muestra información útil acerca de la página pero puede no ser necesaria para todos.

Barra lateral

La barra lateral puede mostrar sus marcadores o el historial de navegación. Puede siempre cerrar la barra lateral haciendo clic en la X de la esquina superior derecha.

Marcadores

Muestra la barra lateral de marcadores, que mantiene sus marcadores visibles todo el tiempo. Ésto puede ser muy útil si usa los marcadores a menudo.

Historial

Muestra la barra lateral de historial, que mantiene un registro de los sitios que se han visitado. Para cambiar cuantos días se deber recordar un sitio, seleccione &pref.menuPath; y busque el panel de Privacidad.

Detener

Deja de cargar la página actual que se está visitando. Es lo mismo que hacer clic en el botón Detener de la barra de herramientas.

Recargar

Obtiene la versión más actualizada de la página actual. Es lo mismo que hacer clic en el botón Recargar de la barra de herramientas.

Tamaño del texto

Si el texto de una página web es demasiado chico o demasiado grande, se puede cambiar a un tamaño más confortable.

Aumentar

Muestra el texto en la página un tamaño más grande. También se puede hacer ésto con el ratón (ver Atajos de ratón para ver las instrucciones).

Disminuir

Muestra el texto en la página un tamaño menor. También se puede hacer ésto con el ratón (ver Atajos de ratón para ver las instrucciones).

Normal

Muestra el texto en la página en el tamaño normal.

Estilo de la página

Permite seleccionar un estilo para la página actual. Por defecto, se selecciona "Estilo básico de página" a menos que el autor especifique un esilo predeterminado diferente.

Sin estilo

Remueve el formato de estilo de la página.

Estilo básico de página

Muestra la pá con un estilo general especificado por el autor.

Si el autor a especificado estilos adicionales de página, &brandShortName; los mostrará aquí para seleccionarlos.

Codifiación de caracteres

Permiete cambiar manualmente la codificación de caracteres de una página web. &brandShortName; normalmente hace esto automáticamente.

Código fuente

Muestra el código fuente del documento actual.

Pantalla completa

Muestra la ventana a pantalla completa. Si tiene la barra de navegación oculta, siempre puede restaurar la ventana presionando F11.

Ir

El menú Ir contiene una lista de los sitios web más recientemente visitados.

Atrás

Navega hacia atrás una página en el historial. Es lo mismo que hacer clic en el botón Atrás en la barra de navegación.

Adelante

Navega hacia adelante una página en el historial. Es lo mismo que hacer clic en el botón Adelante en la barra de navegación.

Inicio

Muestra la página de inicio. Es lo mismo que hacer clic en el botón Inicio en la barra de navegación. Para cambiar la página de inicio, seleccione &pref.menuPath; y abra el panel General. Se puede especificar que se use una página en blanco, la página actual (o el grupo de pestañas), un marcador, o una dirección ingresada manualmente.

Marcadores

Este menú contiene todos sus marcadores guardados.

Marcar esta página...

Agrega la página actual a sus marcadores. Se mostrará un diálogo, permitiendo seleccionar un título para el marcador y especificar donde lo quiere guardar.

Administrar marcadores...

Muestra el administrador de marcadores, una ventana donde se pueden modificar sus marcadores. Puede ordenar, renombrar y cambiar las propiedades de sus marcadores como agregar, remover y moverlos.

Herramientas

Búsqueda web

Resalta la caja de búsqueda web. Allí puede tipear los términos de búsqueda que desee buscar en la Web.

Leer correo (solamente Windows)

Abre su programa de correo electró así usted puede leerlos.

Nuevo mensaje... (solamente Windows)

Abre una ventana de redacción de correo electrónico usando su programa predeterminado para poder enviar un mensaje a alguien.

Descargas

Abre el administrador de descargas, donde se pueden ver las descargas actuales, y también las finalizadas.

Extensiones

Abre el administrador de extensiones, donde se pueden ver todas las extensiones instaladas en esta versión de &brandShortName;. También puede elegir desinstalar extensiones desde este diálogo. Para mayor información, vea el tópico de ayuda sobre Extensiones.

Temas

Abre el administrador de temas. Desde ahí, se puede cambiar el tema actual que &brandShortName; está usando. Se pueden instalar y desinstalar los temas desde este diálogo. Para mayor información, vea el tópico de ayuda sobre Temas.

Consola JavaScript

Abre la consola JavaScript, que rastrea problemas con el código JavaScript. JavaScript es un lenguaje de programación comunmente usado para construir páginas web. Los programadores usan JavaScript para hacer que las páginas sean más interactivas; JavaScript es normalmente usado para validar formularios dinámicamente y seleccionar botones.

Información de la página

Muestra información extensiva sobre la página web actual, como el tipo de documento, codificación, tamaño, e información de seguridad. El diálgo también muestra un listado de los medios y enlaces usados en la página.

Limpiar rastros de &brandShortName;...

Borra los ítems que estén seleccionados en el diálogo de opciones de limpiar rastros en el panel de Privacidad de &pref.menuPath;. Muestra un diálogo de confirmación por defecto.

&pref.pluralCaps;

Muestra la ventana de &pref.pluralCaps;, donde se pueden cambiar muchas &pref.plural; de &brandShortName;.

Ventana

Minimizar

Minimiza la ventana actual.

Zoom

Agranda o restaura la la ventana actual.

(Ventanas abiertas)

Muestra todas las ventanas abiertas por su título.

Ayuda

Contenido de la ayuda

Para usuarios de Internet Explorer (solamente Windows)

Abre la ventana de ayuda mostrando información que puede ayudar a los usuarios de Internet Explorer en su transición a &brandShortName;.

Promocionar &brandShortName;

Abre una página web que brinda información sobre como promocionar &brandShortName;.

Muestra un diálogo con información sobre &brandShortName;, incluyendo la versión actual y una breve lista de créditos.
