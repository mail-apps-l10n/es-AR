Navegando páginas web

Visualizando su página de inicio

Cuando se inicia &brandShortName;, verá su página de inicio. Por defecto, verá la página de inicio de &brandShortName;

Trucos:

  • Para mostrar más contenido web sobre la pantalla, puede usar el modo de Pantalla Completa. El modo de pantalla completa condensa la barra de herramientas de &brandShortName; en una barra de herramientas pequeña. Para habilitar el modo de pantalla completa, simplemente seleccione Ver > Pantalla completa o presione F11.
  • Para ir a su página de inicio rápidamente, presione Alt+Inicio.

Moviéndose a otra página

Puede moverse a una nueva página web ingresando su dirección de Internet o URL en la barra de direcciones. URLs normalmente comienzan con "http://" seguido por uno o más nombres que identifican la dirección. Un ejemplo es "http://www.mozilla.org/".

  1. Haga clic en la barra de direcciones para seleccionar la URL que ya se encuentra allí.
  2. Tipee la URL de la página que quiera visitar. La URL que tipee reemplaza cualquier texto que se encuentre en la barra de direcciones.
  3. Presione Enter.

Truco:Para seleccionar rápidamente la URL de la barra de direcciones, presione &accelKey;+L.

¿No sabe una URL? Intente tipear una palabra general (por ejemplo "regalos" o "flores") en la barra de búsqueda y pulse Intro. Esto comenzará una búsqueda web para el asunto que tipeó y le dirá la URL que ud. está buscando

Haciendo clic en un enlace

La mayoría de las páginas web contienen enlaces que en los que puede hacer clic para moverse a otras páginas.

  1. Mueva el puntero del mouse hasta que cambie a un dedo apuntando. Esto sucede siempre que el puntero esté sobre un enlace. La mayoría de los enlaces son texto subrayado, pero los botones y las imágenes también pueden ser enlaces.
  2. Haga clic en el enlace una vez. Mientras la red localiza la página del enlace, los mensajes de estado aparecerán en la parte inferior de la ventana.

Retracing Sus Pasos

Hay varias maneras de revisitar las páginas:

  • Para retroceder o avanzar una página, haga clic en el botón Atrás o Adelante.
  • Para retroceder o avanzar más de una página, haga clic en el pequeño triángulo sobre los botones Atrás y Adelante. Verá una lista de páginas que ha visitado recientemente; para volver a una página, elíjala de la lista.


  • Para ver una lista de cualquiera de las URLs que haya tipeado en la barra de la direcciones, haga clic en la flecha en el extremo derecho de la barra de direcciones. Para ver una página, elíjala de la lista.


  • Para elegir una de las páginas que ha visitado durante la sesión actual, abra el menú Ir y utilice la lista en la sección inferior del menú
  • Para elegir una de las páginas que usted ha visitado durante las sesiones pasadas, abra el menú Ir y elija Historial. Verá la barra lateral de Historial. La barra lateral de Historial exhibe una lista de carpetas. Con doble clic, las carpetas exhiben subcarpetas o marcadores a las páginas web. Usted puede hacer doble clic en la URL al lado del icono del marcador para ver esa página.

Parando y recargando

Si una página carga demasiado despacio o no desea ver más una página, pulse el botón Parar.

Para recargar la página actual o conseguir la versión más actualizada, haga clic en el botón Recargar o presione &accelKey;+&shiftKey;+R.

Pestañas de Navegación

Cuando usted visita más de una página Web a la vez, usted puede usar Pestañas de Navegación para navegar la Web más rápido y más fácil.

Las Pestañas de Navegación le deja abrir pestañas, cada una desplegando una página Web, dentro de una sola ventana &brandShortName;. Usted no tiene que tener varias ventanas abiertas para visitar varias p঩nas Web diferentes. Esto libera el espacio sobre su escritorio. Usted puede abrir, cerrar, y recargar páginas Web cómodamente en un lugar sin la necesidad de cambiar a otra ventana.

Para más Info, vea Pestañas de Navegación.

Usando la Barra Lateral

La Barra Lateral es un área sobre el lado izquierdo de la pantalla que usted puede usar para la inspección de marcadores o historial. Las extensiones pueden agregar nuevas maneras de usar la Barra Lateral también.

Para ver un item en la Barra Lateral, seleccione Ver > Barra Lateral.Desde allí usted puede seleccionar la etiqueta de la Barra Lateral que usted quiera.

Buscar

Por ejemplo, si usted quiere encontrar la información sobre muñecas de bebé:

  2. Tipee la frase muñeca de bebé. Su tipeo sustituye cualquier texto que haya actualmente en la barra de búsqueda.
  3. Presione &enterKey; para buscar.

Los resultados para "muñeca de beb砠aparecen en la ventana &brandShortName;

Para buscar palabras seleccionadas en una Página Web

&brandShortName; permite que usted busque palabras que usted selecciona dentro de una página web:

  1. Seleccione (resalte) cualquier palabra en una página Web.
  2. Click-derecho Presione &ctrlKey;,clickee el botón del mouse, y elija "Búsqueda web para [su palabra seleccionada]" desde el menú.

&brandShortName; abre una nueva ventana y use su motor de búsqueda por defecto para buscar sus palabras seleccionadas

Buscar Dentro de una Página

Para encontrar el texto dentro de la página que usted está viendo actualmente en &brandShortName;:

  1. Seleccione Editar > Buscar en esta página. Si la página actual contiene marcos, usted puede necesitar clickear dentro de un marco antes de que usted comience su búsqueda. Una nueva barra de herramientas debe aparecer cerca del fondo de su navegador.
  2. Tipeeí el texto que usted desea encontrar
  3. Elija de las opciones siguientes de búsqueda:
    • Buscar Siguiente:encuentra el texto en la página que está debajo de la posición actual del cursor.
    • Buscar Anterior:encuentra el texto que está arriba de la posición actual del cursor
    • Resaltar:resalta ocurrencias de su cadena de búsqueda en la página actual.
    • Coincidir Mayúsculas: limita la búsqueda al texto que tiene la misma capitalización que su cadena de búsqueda.
  4. La búsqueda automáticamente comienza cuando algo es tipeado en el cuadro de texto.

Encontrar la misma palabra o frase otra vez:

  • Seleccione Editar > Buscar de nuevo.

Copiar, Guardar, e Imprimir Páginas

Copiar parte de una página

Copiar texto desde una página:

  1. Seleccione el texto.
  2. Seleccione Editar > Copiar desde la barra de menú.

Ud. puede pegar el texto en otros programas.

Copiar un enlace (URL) o una imagen de enlace desde una página:

  1. Posicione el puntero sobre el enlace o imagen
  2. Right-clickPresione &ctrlKey; y clickee sobre el enlace o imagen para desplegar un menú emergente.
  3. Elija Copiar Dirección del Enlace o Copiar Dirección de la Imagen. Si una imagen es también un enlace, ud. puede elegir una opción o la otra del menú.

Ud. puede pegar el enlace en otros programas o en la barra de herramientas de &brandShortName;

Guardar todo o parte de una página

Para guardar una página entera en &brandShortName;:

  1. Seleccione Archivo > Guardar página como. Ud. verá el cuadro de diálogo Guardar como.
  2. Elija una posición para la página salvada
  3. Elija un formato para la página que ud. quiere salvar:
    • Página Web Completa: Guarda la página web entera a lo largo con imágenes. Esta opción permite que usted lo vea como al principio mostrado con imágenes, pero esto puede no guardar la estructura del enlace de HTML de la página original. &brandShortName; crea un nuevo directorio donde la página es guardada con imágenes y otros archivos necesarios para mostrar la página Web entera.
    • Página Web, sólo HTML : Guarda la página original sin imágenes. Esta opción conserva la estructura del enlace de HTML original en un archivo.
    • Archivos de texto:Guarda la página original como un archivo de texto. Esta opción no conservará la estructura del enlace de HTML original, pero permitirá que usted vea una versión de texto de la página Web en cualquier editor de texto.
  4. Tipee un nombre de archivo para la página y clickee Guardar.

Guardar un marco desde adentro de una página web:

  1. Posicione el puntero del mouse adentro del marco.
  2. Presione &ctrlKey; y clickee sobreClick-Derecho sobre el marco para desplegar un menu popup.
  3. Seleccione Este Marco > Guardar Marco como desde el submenú. Ud. verá el cuadro de diálogo Guardar Como.
  4. Elija una posición para la página guardada.
  5. Elija un formato para la página que usted quiere salvar.
  6. Tipee un nombre de archivo para la página y clickee Guardar.

Guardar un archivo en su disco duro le deja ver la página cuando usted no está conectado a Internet

Guardar una imagen de una página:

  1. Posicione el puntero del mouse sobre la imagen.
  2. Click-DerechoPresione &ctrlKey; y clickee sobre sobre la imagen para desplegar un menú popup
  3. Seleccione Guardar Imagen Como. Ud. verá el Cuadro de diálogo Guardar Imagen.
  4. Elija una posición para la imagen guardada.
  5. Tipee un nombre de archivo para la imagen y clickee Guardar.

Guardar una página sin desplegarla (que es útil para recuperar a una página sin formatear que no es querida para verla):

  1. Posicione el puntero del mouse sobre el enlace a la página.
  2. Click-DerechoPresione &ctrlKey; y clickee sobre el enlace para desplegar un menú popup.
  3. Seleccione Guardar Enlace como.... Ud. verá el Cuadro de diálogo Guardar Como.
  4. Elija una posición para la página guardada.
  5. Tipee un nombre de archivo para la imagen y clickee Guardar.

Importante:Algunos enlaces automáticamente descargan y guardan archivos en su disco duro después que ud. los clickea. Las URLs para estos enlaces frecuentemente comienzan con "ftp" o terminan con una extensión de tipo de archivo como "au" or "mpeg." Estos enlaces podrían transmitir software, sonido, o archivos de película y pueden lanzar aplicaciones de ayuda que soportan los archivos.

Truco: Para colocar una imagen como su fondo de pantalla en Windows, haga click-derecho sobre una imagen y elija Poner como fondo de pantalla desde el menú popup. You can choose various display &pref.plural; through the dialog that is shown.

Imprimiendo una Página

Imprimir la página actual:

  • Seleccione Archivo > Imprimir.

Imprimir texto seleccionado:

  • Seleccione el texto en la página actual.
  • Seleccione Archivo > Imprimir. El cuadro de diálogo Imprimir aparecerá
  • Debajo de Intervalo de Impresión, clickee Selección.

The web page's author and the size of the printed page, not the size of the onscreen window, determine placement of content on the printed page. Text is wrapped and graphics are repositioned to accommodate paper size.

Usando Vista Previa

Para tener una mirada temprana de cómo se verá una página antes de imprimirla, ud. puede usar Vista Previa:

  • Seleccione Archivo > Vista Previa.

En Vista Previa, ud. puede hacer lo siguiente con las páginas que ud. desea imprimir:

  • Ver una Vista Previa de cada página: Clickee el botón Siguiente , Previo , Primero , o Ultimo para moverse entre las páginas.
  • Cambie la escala (tamaño) de las páginas: En el menú desplegable Escala elija "50%" para hacer cada página de la mitad del tamaño de la página original. Para cambiar automáticamente el tamaño a las páginas de modo que encajen en el ancho del papel, elija "Ajustar a la hoja". Ud. puede también tipear su propio porcentaje eligiendo "Personalizar...".
  • Cambiar la orientación de la página: Elija Normal para colocar la página con el lado más corto hacia arriba. Elija Apaisada para colocar la página de costado con el lado más largo hacia arriba.
  • Ir a Configurar Página: Clickee Configuración de Página para efectuar configuraciones de las páginas que quiera imprimir
  • Imprimir: Clickee Imprimir para imprimir las páginas

Nota: Algunas funciones de Vista Previa son diferentes o inexistentes en Mac OS y Linux.

Usando Configuración de Página

Note: Algunas funciones de la Configuración de Páginas son diferentes o inexistentes en Mac OS y Linux.

Para configurar cómo serán impresas las páginas en &brandShortName;, ud. puede usar Configuración de Página:

  • Seleccione Archivo > Configuración de Página.

En Configuración de Página, ud. puede cambiar los siguientes parámetros para las páginas que ud. desea imprimir:

  • Formato & Opciones: Elija la orientación, escala y otras opciones:
    • Orientación:
      • Normal: Elija ésta para posicionar la página en forma normal, con el lado más corto hacia arriba.
      • Apaisada: Elija ésta para posicionear la página de lado, con el lado más largo hacia arriba.
    • Escala: Tipee adentro un porcentaje del tamaño original. Por ejemplo, tipee "50" para hacer cada página de la mitad del tamaño de la página original.
      • Ajustar al ancho de Página: Seleccione esto para redimensionar la página al tamaño del ancho del papel.
    • Opciones:
      • Imprimir Fondo (colores e imágenes): Seleccione esto para imprimir imágenes y colores de fondo. Si no está seleccionado solamente los cuadros y el color en el primero plano (en frente) se imprimen.
    • Márgenes & Encabezado/pie: Clickee esta pestaña para configurar márgenes, encabezado y pie de página:
  • Márgenes:
    • Superior, Inferior, Izquierda, Derecha: Tipee un márgen en las unidades dadas para los márgenes superior, inferior, izquierdo y derecho.
  • Encabezado & Pie: Cada lista drop-down representa un encabezado o un área del pie. La fila superior de listas drop-down está para las áreas izquierda, de centro, y derechas del encabezado. La fila inferior está para las áreas izquierdas, de centro, y derechas del pie. En cada lista drop-down, elija una de las opciones siguientes:
    • --blank--: No muestra nada en esta área.
    • Título: Muestra el título de la página web.
    • URL: Muesra la URL de la página web (Las URL usualmente comienzan con "http://").
    • Fecha/Hora: Muestra la fecha y hora cuando la página web es impresa.
    • Página #: Muestra el número de página de cada página.
    • Página # de #: Muestra el número de página junto con el número total de páginas. Por ejemplo, si ud. imprime una página web de 5 páginas, "3 de 5" debería ser mostrado en la tercer página.
    • Personalizar: Tipee su propio texto. Ud. puede incluir cualquiera de los siguientes códigos para imprimir la información específica:
      • &PT: Página Número con Total (Ejemplo: "3 of 5")
      • &P: Página Número
      • &D: Fecha
      • &U: URL
      • &T: Título de Página

Truco:Para ver una vista previa de los cambios hechos a la Configuración de página, use Vista Previa.

Mejorar Velocidad y Eficiencia

Manejando Diferentes Tipos de Archivos

&brandShortName; puede manejar muchos tipos de archivos. Sin embargo, para algunos archivos tal como películas o música, &brandShortName; necesita plug-ins o "aplicaciones auxiliares" que puedan manejar estos archivos. Si &brandShortName; no tiene la ayuda necesaria desde otra aplicación o agregado, todavía puede guardar el archivo en su disco rígido. Cuando guarde los archivos, ud. puede guardar la pista de ellos usando el Administrador de Descargas.

Plug-ins
Plug-ins son aplicaciones auxiliares que amplían la funcionalidad dentro de &brandShortName; y corren dentro de él. Plug-ins tal como Sun Java, Macromedia Flash, y RealNetworks RealPlayer permiten que &brandShortName; muestre archivos multimedia y ejecute pequeñas aplicaciones tales como películas, animaciones y juegos.

Ud. puede también instalar plug-ins después de que usted instale el &brandShortName;. Para ver los plug-ins para &brandShortName; que están instalados actualmente, seleccione &pref.menuPath;. Seleccione la categoría Descargas y clickee el botón Plug-Ins.... Usted puede también habilitar o inhabilitar plug-ins actualmente aquí instalados.

Para ver una lista completa de los plug-ins para &brandShortName; que usted puede instalar, vea la Página de plug-ins del navegador en Netscape.com.

Aplicaciones Auxiliares
Cuando los archivos no se pueden abrir dentro de &brandShortName;, usted puede elegir lanzar las aplicaciones auxiliares que se abren fuera de &brandShortName;. Por ejemplo, para escuchar archivos MP3, los programas como Winamp se pueden abrir fuera de &brandShortName;.

Los ajustes en el panel Descargas en la ventana &pref.pluralCaps; le dice a &brandShortName; cuáles aplicaciones auxiliares va a utilizar en su computadora. En la sección Tipos de Archivos, ud. puede ver cómo diversos tipos de archivo serán manejados por el &brandShortName;. Usted puede cambiar la acción para un tipo de archivo o quitarla. También, si &brandShortName; no sabe cuál aplicación auxiliar usar, éste le permite encontrar la aplicación auxiliar dentro del cuadro de diálogo de descargas.

El cuadro de diálogo de descarga tiene estas opciones:

  • Abrir Usando: &brandShortName; abre el archivo usando el programa definido por defecto en sus ajustes del sistema. Usted puede también elegir un programa distinto seleccionando Otro... en la lista drop-down.
  • Grabar a Disco: &brandShortName; le permite a ud. seleccionar una ubicación para grabar el archivo. Si ud. tiene seleccionado Guardar todos los archivos en esta carpeta en el panel Descargas de &pref.pluralCaps;, los archivos serán grabados automáticamente en la carpeta predeterminada.
  • Ejecutar la acción asociada automáticamente con cada uno de los siguientes tipos de archivo: Cuando esto está seleccionado, &brandShortName; automáticamente ejecutará la acción seleccionada la próxima vez que ud. descargue un archivo de ese tipo.

Truco: Ud. puede borrar su &pref.plural; grabada para Ejecutar la acción asociada automáticamente con cada uno de los siguientes tipos de archivo en el panel Descargas de &pref.pluralCaps;.

Haciendo &brandShortName; su Navegador Predeterminado

Cambiando los ajustes del Cache

&brandShortName; almacena copias de páginas con frecuencia alcanzadas en el cache. Haciendo esto &brandShortName; no tiene que recuperar la página de la red cada vez que usted vea la página web.

Para fijar el tamaño del cache o borrar éste (Windows y Unix solamente):

  1. Seleccione &pref.menuPath;.
  2. Debajo de la categoría Privacidad , clickee Cache para expandir el listado.
  3. Ingrese un número en el campo Usar hasta para especificar el tamaño del caché. 50MB a 70MB es un buen tamaño. Para borrar el cache inmediatamente, clickee el botón Borrar a la derecha de la categoría Cache.

Importante:Un cache más grande permite que más datos sean recuperados rápidamente, pero más espacio de su disco duro se utiliza.

Cuando ud. quita &brandShortName;, éste realiza el mantenimiento del caché. Si el mantenimiento dura más de lo que ud. desea, trate reduciendo el tamaño del cache de disco.

Para recargar una página en cualquier momento:

  • Clickee el botón Recargar en la Barra de Navegación de &brandShortName;. La computadora comprueba en Internet para cerciorarse de que usted tiene la última versión de la página.
